# Hey! I'm Filing Here

In this lab, I successfully implemented the ext2 file system, specifcally dealing with 2 directories, 1 regular file ,and a symbolic link. 

## Building

To build the program:

'
make
'

## Running

To run this program:

'
./ext2-create 
'


## Cleaning up

To clean up, run:

make clean
